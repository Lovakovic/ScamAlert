Why the API Key?

To analyze the URLs and ensure your safety, our extension relies on the VirusTotal service. In order to use it, we need you to register for a free account with VirusTotal and provide the API key to the extension.

Steps to Get Your API Key

  1. Register for a free account on VirusTotal.
  2. Once registered, go to your API key section or use the link https://www.virustotal.com/gui/user/USERNAME/apikey replacing USERNAME with your VirusTotal's account username.
    This image guides you to access the API Key section of your VirusTotal account.
  3. Click the `Copy API key` button to copy the API key to your clipboard.
    This image shows you where to click to copy the API key to your clipboard.

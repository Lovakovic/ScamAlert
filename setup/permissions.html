Permissions & Privacy

This extension requires certain permissions to function properly. These are:

  • Active Tab: This allows the extension to interact with the current active tab and scan the URLs for potential threats.
  • Web Request: This permission lets the extension monitor web requests made by the browser, such as when you navigate to a new URL. This is crucial for determining when to scan a URL.
  • Local Storage Access: This is used to store settings, the API key, and cache scan results for better performance.
  • Unlimited Storage: The extension can store an unlimited amount of data in the browser storage. This ensures that the cached scan results and settings aren't accidentally deleted by the browser when it's freeing up space.
  • Alarms: This allows the extension to manage and schedule certain tasks within the extension, like cleanup of old scan data or fetching of analysis results for domains.
  • Host permissions: This ensures the extension has the ability to scan and analyze content across all websites you visit.

We only send URLs to the VirusTotal API for analysis, and do not share your data with anyone else. Your privacy is our utmost priority.
